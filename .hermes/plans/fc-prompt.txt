## Task 1: 创建 sql/tools.py
新建文件 sql/tools.py，使用 LangChain 1.x 的 @tool 装饰器创建一个 create_sql_tool() 工厂函数。

要求：
- 内部延迟导入 SQLQueryAgent
- 工具名: sql_query
- 工具描述要包含适用/不适用场景（让LLM知道何时调用）
- 输入参数: question (str)
- 返回: 查询结果的字符串（成功时包含 SQL 语句）
- 关键代码模式:
```python
from langchain_core.tools import tool
from sql.agent import SQLQueryAgent

def create_sql_tool(db_uri=None):
    agent = SQLQueryAgent(db_uri=db_uri)
    
    @tool(description="详细的工具描述...")
    def sql_query(question: str) -> str:
        result = agent.query(question)
        if result["success"]:
            ...
        else:
            ...
    
    sql_query.name = "sql_query"
    return sql_query
```
